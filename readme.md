# HUTCourse 湖南工大课程助理

## 一、项目核心内容

本项目是一款面向湖南工业大学（HUT）学子的**智能校园助手 Android 应用**，以「课表管理」为核心，融合 AI 大模型能力，打造覆盖学业、生活、日程的一站式校园智能服务平台。应用支持强智教务系统课表自动抓取与解析、智能日程管理、校内导航、空教室查询、天气服务及联网信息检索，并通过自然语言对话实现全功能语音/文字交互。

## 二、AI 技术应用方向

- **大模型智能体（LLM Agent）**：基于 System Prompt 编排与多轮工具调用（ReAct 模式），AI 助手可自主决策调用课表查询、日程管理、校内导航、空教室查询、联网搜索等 6 大技能，实现复杂任务的自动分解与执行。
- **多模态交互**：支持文本与图片混合输入，用户可通过上传图片或自然语言描述，让 AI 自动解析并创建日程、识别课程信息。
- **动态技能编排**：通过前端 Matter 技能索引与开关配置，实现 AI 能力的模块化加载与热插拔，灵活适配不同使用场景。
- **上下文感知对话**：集成时间语义解析、周次计算、位置感知，AI 能准确理解"明天下午在哪上课""这附近有什么空教室"等校园场景化提问。

## 三、与专业技能融合点

- **移动应用开发（Android）**：采用 Java + Material Design 3 构建原生应用，运用 ViewPager2、BottomSheet、Fragment 等组件实现流畅的多页面交互。
- **数据解析与存储**：使用 Jsoup 解析教务系统 HTML 课表，SQLite 实现本地课程与日程持久化，OkHttp 处理网络请求与 Cookie 会话保持。
- **地理信息处理**：集成高德地图 SDK，实现校内建筑坐标管理、位置语义化（楼内/附近/两楼之间）与导航跳转。
- **软件工程与架构设计**：采用 Manager 模式封装技能模块（Course/Agenda/Navigation/Classroom/Note/Search），通过 SkillCommandCenter 统一调度，实现高内聚低耦合。

## 四、创新亮点

1. **教务系统深度对接**：突破强智教务系统无开放 API 的限制，通过 Cookie 模拟登录 + HTML 解析实现课表自动同步，并支持实验课表合并与备注课程解析。
2. **AI 驱动的自然语言课表交互**：无需手动翻查课表，用户通过对话即可查询"今天还有什么课""下周三的高数在哪上"，AI 自动完成日期计算、周次匹配与结果返回。
3. **校园场景化工具链**：将课表、日程、导航、空教室、天气、笔记整合为 AI 可调用的技能矩阵，形成"查询—规划—导航—记录"的闭环体验。
4. **智能日程生成**：用户仅需用自然语言描述（如"周五下午三点去图书馆开会"），AI 自动解析时间、地点并创建带重复规则与优先级的日程项。

## 五、预期应用效果

- **提升学生日常效率**：将课表查询、教室寻找、日程规划等碎片化操作收敛为"一句话搞定"，预计日均节省操作时间 10–15 分钟。
- **降低信息获取门槛**：新生与校外人员可通过自然语言快速熟悉校园布局、课程安排与公共资源分布。
- **可扩展的校园智能中台**：技能模块化架构支持后续快速接入图书馆座位查询、考试成绩分析、校园二手交易等新能力，具备向其他高校推广的通用性。
