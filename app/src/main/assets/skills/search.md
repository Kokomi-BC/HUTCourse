# Search Skill

## Purpose

- 处理依赖实时互联网信息的问题。
- 通过 Tavily 检索外部网页并返回可引用摘要。

## Commands

- `tavily.search <关键词>`: 发起联网检索。

## Query Rules

- 关键词应简洁且包含核心实体（人名、地点、事件、时间）。
- 当问题是校内静态信息时，优先校内技能，不要滥用联网检索。
- 若返回为空或报错，明确告知未检索到结果或检索失败，不要编造。

## Response Style

- 先给出结论，再列关键信息来源。
- 来源链接应简短可读，避免堆叠重复结果。
